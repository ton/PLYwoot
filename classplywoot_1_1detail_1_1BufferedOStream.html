Detailed Description

Wrapper around some output stream that provides buffered output functionality. This will always buffer some compile-time given size of bytes up front, and data is written to this buffer first until either the buffer fills up or an instance of this object goes out of scope, at which point all buffered data it is written to the wrapped output stream. This improves writing to file-backed output streams considerably.
