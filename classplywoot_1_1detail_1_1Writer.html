Detailed Description

template<typename FormatWriterPolicy>
class plywoot::detail::Writer< FormatWriterPolicy >

Represents a generic PLY writer that is parameterized with format specific functionality through the FormatWriterPolicy type, which should adhere to the following model requirements:

  • template<typename T> void writeNumber(T t);
  • template<typename PlySizeT, typename PlyT, typename SrcT> void writeList(const SrcT *t, std::size_t n) const;
  • template<typename PlyT, typename SrcT> void writeNumbers(const SrcT *t, std::size_t n) const;
  • void writeNewline();
  • void writeTokenSeparator();
