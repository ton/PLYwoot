Detailed Description

Wrapper around some input stream that provides buffered input functionality. This will always buffer some compile-time given size of bytes up front, and data is read from this buffer until the buffer is exhausted, at which point it is refilled again with the next block of data from the wrapped input stream. This improves reading from file-backed input streams considerably.
